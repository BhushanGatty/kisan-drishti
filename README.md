 # Kisan-Drishti 
*Vernacular-First Agentic Mission Control for Autonomous Agriculture*

##  Overview
Kisan-Drishti is an AI-powered "Mission Control" that allows rural farmers in Bharat to operate autonomous strawberry-harvesting robots using simple voice notes in their native tongue via WhatsApp. 

##  The Prototype
Our solution bridges the gap between complex robotics and non-technical users. 
- **The Robot:** Hello Robot Stretch 2 (MuJoCo Menagerie model).
- **The Brain:** Google Antigravity Agent for autonomous task planning.
- **The Bridge:** n8n workflow translating vernacular audio into robot intents.

##  Key Features
- **Zero-UI Accessibility:** Control advanced robotics through the most familiar interface: WhatsApp.
- **Multi-lingual Support:** Transcribe and translate major Indian dialects (Hindi, Marathi, Kannada, etc.).
- **Agentic Orchestration:** The robot re-plans paths autonomously if it encounters obstacles.
- **Physics-Validated Picking:** Real-time simulation in MuJoCo ensures collision-free strawberry harvesting.

##  Tech Stack
- **AI Agent:** Google Antigravity (Gemini 1.5 Pro)
- **Orchestration:** n8n
- **Physics Engine:** MuJoCo
- **Robot Model:** Hello Robot Stretch 2
- **Communication:** WhatsApp Business API
